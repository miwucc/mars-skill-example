本次迭代开发一个简单的静态 HTML 页面。

- 页面初始状态展示一个搜索框和一个搜索按钮。
- 本迭代不做真实搜索，不调用后端或外部接口，不解析输入文章内容。

- 输入内容不影响展示结果。
